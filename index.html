uPDF Joiner is designed to simplify merging, splitting, and extracting PDF pages in a lightweight way. Supports Windows Vista, 7, 8, 8.1, 10, and 11.

Getting Started & Shortcuts

  1. Add PDF Files [CTRL + O]
    • Add PDF files to merge, split, or extract.
  2. Merge PDF Files [CTRL + M]
    • Merge multiple PDFs into one.
    • Reorder with [Up] and [Down] buttons.
  3. Extract Pages [CTRL + E]
    • Extract specific pages.
    • Examples:
      • Single: 3
      • Multiple: 2,4,8
  4. Split PDF Files [CTRL + S]
    • Split a PDF into individual pages.
  5. Remove [DEL]
    • Remove selected PDF from the list.
  6. Clear All [CTRL + R]
    • Clear all files from the list.
